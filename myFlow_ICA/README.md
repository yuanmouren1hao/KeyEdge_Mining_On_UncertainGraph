# myFlow_ICA Project

> 该项目是基于流量和可靠性的不确定图关键边评估的增量算法。保存之前的中间过程，将边进行分类分别计算，在排序的时候分别计算。
> 三个指标分别是：最大流>分布可靠性>网络可靠性

## Step

 >1. 首先计算不确定图G中的每一条边断掉之后能够达到的最大流，并从小到大排序；
 >2. 在原不确定图G上做最可靠最大流分布，利用状态划分算法SDBA获取所有满足最大流的子图区间；
 >3. 将不确定图G中的边定性分类，分别为ABC类边。其中A类边存在于所有子图中，发生故障最大流减少；B类边存在于部分满足最大流子图中，发生故障最大流不变，但是分布可靠性和容量可靠性可能发生改变；C类边断掉对不确定图不会造成影响；
>4. 根据不同类别的边进行不同方式的计算。其中A类边重复计算，B类边增量计算，C类边不需要计算；
>5. 设置一个滑动窗口，大小为2，按照计算的最大流进行滑动，i与i+1， 如果i与i+1不相等，则i不需要计算，
